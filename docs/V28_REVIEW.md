# PAOPAO V28 REVIEW

更新日: 2026-07-11

## 目的

V28は大規模な方向転換ではなく、V27までの構成を維持した品質安定化版です。

## 変更

- HOME / WORK / EXPERIENCE / PROCESS / CURRENT / JOURNAL / CONTACTのページ分割を維持
- 見出しの句点を削除し、現代のビジネスサイトとして自然な日本語へ修正
- ヘッダーナビへ日本語補助ラベルを追加
- ページごとの意味を示す低彩度のシステム図を背景へ追加
- WORKに「課題収集 → 整理 → 手順化 → 運用」の変換図を追加
- EXPERIENCEに3つの経験領域を示す軸を追加
- PROCESSに「確認 → 修正 → 更新」の小さな改善ループを追加
- CURRENTにLEARNING / BUILDING / TESTINGの現在地を追加
- JOURNALにCAPTURE / REVIEW / UPDATEの記録目的を追加
- カードやケースへ控えめな反応線を追加
- JOURNALの旧形式・新形式を同じ一覧で扱う互換処理を追加
- V28のCSS・JSを既存ランタイムへ追加し、これまでのページ切り替えや管理機能を維持
