# PAOPAO RELEASE CHECKLIST

公開前に、該当しない項目を除いてすべて確認する。確認していない項目を「問題なし」と扱わない。

## A. 事実・文章

- [ ] 年齢、地域、経歴、期間、数値が承認済みの事実と一致する
- [ ] 実績と現在進行中の活動が区別されている
- [ ] 架空の顧客、案件、口コミ、資格、売上、納期を追加していない
- [ ] 見出しだけで要点が分かる
- [ ] 抽象的なビジネス用語を具体的な担当・成果物へ置き換えた
- [ ] 現代日本語として声に出しても不自然でない
- [ ] 教材、自己啓発、企業パンフレットのような口調が残っていない
- [ ] 誤字、脱字、助詞、表記揺れ、句読点を確認した

## B. 情報設計

- [ ] HOMEで人物、中心能力、主要実績が短時間で分かる
- [ ] WORKで依頼可能な内容が分かる
- [ ] EXPERIENCEで根拠となる担当・数字が分かる
- [ ] PROCESSで相談後の流れと成果物を想像できる
- [ ] CURRENTを実績と誤認しない
- [ ] JOURNALとCONTACTへ迷わず到達できる
- [ ] HOMEと各詳細ページの説明が重複しすぎていない
- [ ] 全情報を一つの長いページへ戻していない

## C. デザイン

- [ ] 墨色、朱、青緑、生成りの世界観が一貫している
- [ ] 日本要素が安っぽい和柄装飾になっていない
- [ ] 文字階層、余白、グリッドが揃っている
- [ ] 情報量に対して十分な余白と区切りがある
- [ ] 本人写真は1枚で、顔を加工していない
- [ ] CTAにInstagramロゴがあり、リンク先と一致する
- [ ] 不要なカード化や枠の乱用がない
- [ ] 装飾が本文の理解を妨げていない

## D. モーション・メディア

- [ ] オープニングが中央の斬撃から連続的に開く
- [ ] ページ切り替え演出が短く、二重発火しない
- [ ] 背景動画が無音・ループ・playsinlineで動く
- [ ] 動画が失敗しても背景画像で成立する
- [ ] スマホ・低負荷・reduced motionで適切に軽量化される
- [ ] 動画や画像の上で文字が読める
- [ ] レイアウトシフトや白いフラッシュがない
- [ ] メディア容量が不必要に巨大化していない

## E. PC表示

確認幅の基準: 1440px / 1024px

- [ ] ヘッダーとナビゲーションが収まる
- [ ] ページ切り替えが動く
- [ ] ジグザグフローの線、番号、注釈が正しく対応する
- [ ] 固定CONTACTが本文へ重ならない
- [ ] 横方向のはみ出しがない
- [ ] ホバーなしでも情報が読める
- [ ] 写真、動画、背景画像が表示される

- [ ] ボタンとリンクを指で押しやすい
- [ ] ジグザグフローが正しい縦順に変形する
- [ ] 固定要素が本文やブラウザUIに重ならない

- [ ] 見出し階層が自然

- [ ] 何を入力するか画面内で説明されている

## リリース判定

次のいずれかが残っている場合は「完成」「修正済み」と断言しない。

- 公開URLを開いていない
- スマホ表示を確認していない
- JOURNALの本番保存を試していない
- コンソールエラーがある
- ユーザーの実機で再現している不具合が未解決
- 秘密情報が露出している

公開報告には、確認済みと未確認を分けて記載する。
